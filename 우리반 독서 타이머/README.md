# 우리 반 독서 타이머 📚

선생님과 아이들이 함께 만들어가는 즐거운 독서 시간! 
우리 반 학생들의 매일매일 독서 시간을 타이머로 측정하고, 기록장에 누적해서 확인할 수 있는 웹 애플리케이션입니다.

## ✨ 주요 기능
- **직관적인 뽀모도로 타이머 UI**: 각 학생별로 독립적인 타이머가 작동하며, 귀엽고 동글동글한 디자인으로 아이들의 흥미를 유발합니다.
- **자동 누적 기록장(📖)**: 매일 독서를 마치고 '리셋' 버튼을 누르면, 단 1초라도 책을 읽은 학생들의 기록이 날짜별로 영구 저장됩니다.
- **철저한 개인정보 보호**: 이름 대신 번호(1번~25번)를 사용하여 개인정보 유출 걱정 없이 학교 전자칠판에서 안전하게 사용할 수 있습니다.
- **독립적인 로컬 저장소**: 브라우저의 `localStorage`를 사용하여 별도의 서버 가입이나 로그인 없이도 기기별로 안전하게 데이터가 저장됩니다.

## 🛠 사용 기술
- **Frontend**: React, Vite
- **Styling**: Vanilla CSS (Beige & Pink Theme, Jua Font)
- **Deployment**: GitHub Pages
- **Icons**: Lucide React
