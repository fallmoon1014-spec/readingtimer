# PRD: My Dev Lab - 학습용 웹앱 포털 & 런처

## 1. 개요
개발자가 직접 만든 웹앱을 한곳에 모아 관리하고, 즉시 실행할 수 있는 개인용 학습 웹포털입니다. DB 없이 JSON 파일로 데이터를 관리하여 유지보수가 간편한 구조를 지향합니다.

## 2. 핵심 목표
- **간편한 관리:** `apps.json` 파일 수정만으로 앱 리스트 업데이트.
- **빠른 실행:** 앱 런처 기능을 통해 원클릭으로 데모 접속.
- **학습 기록:** 각 프로젝트의 기술 스택과 설명을 한눈에 파악.

## 3. 기능 명세
- **앱 목록 관리:** 전체 앱 리스트를 카드 형태로 전시.
- **카테고리 분류:** Utility, Project, Experiment 등 용도별 분류 지원(데이터 구조상 지원).
- **앱 카드 UI:** 앱 이름, 설명, 기술 태그, 실행 버튼 제공.
- **런처 기능:** [실행하기] 버튼을 통해 배포된 라이브 데모로 즉시 이동.

## 4. 기술 스택
- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS
- **Data Source:** `data/apps.json` (로컬 데이터 파일)
- **Deployment:** Vercel (자동 배포)

## 5. 데이터 구조 (apps.json)
```json
[
  {
    "id": 1,
    "title": "앱 제목",
    "category": "분류",
    "tech": ["기술1", "기술2"],
    "demoUrl": "데모주소",
    "repoUrl": "깃허브주소",
    "description": "앱 설명"
  }
]
```

## 6. 개발 단계
1. **환경 구축:** Next.js 프로젝트 생성.
2. **데이터 연결:** `data/apps.json` 생성 및 불러오기 구현.
3. **UI 구현:** `AppCard` 컴포넌트 설계 및 그리드 레이아웃 배치.
4. **배포:** Vercel을 통한 웹 포털 런칭.
